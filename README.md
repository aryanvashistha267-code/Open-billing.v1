## Features

* Fast billing workflow
* Invoice generation
* Thermal print optimized
* GST support
* Product management
* CSV import
* Invoice history
* Stock auto reduction
* Low stock support
* Mobile responsive
* Offline-friendly
* PWA ready
* JSON backup system
* Local browser storage
* Multi-business compatible

## Tech Stack

* HTML
* CSS
* Vanilla JavaScript
* LocalStorage
* PWA

No backend required.

## Storage System

OpenBilling stores data locally in browser storage.

Each device/browser keeps separate data automatically.

No login required.

## Important Warning

Clearing browser data may remove:

* invoices
* products
* settings
* history

Use backup/export regularly.

## GitHub Pages Deployment

1. Create repository

Example:

```txt id="g1"
openbilling
```

2. Upload:

* index.html
* css/
* js/
* assets/
* manifest.json
* sw.js

3. Enable GitHub Pages

Go to:

```txt id="g2"
Repository → Settings → Pages
```

Select:

* Deploy from branch
* main branch
* /root

4. Your site becomes:

```txt id="g3"
https://yourusername.github.io/openbilling/
```
